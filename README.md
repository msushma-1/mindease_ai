# 🌿 MindEase — Your 24/7 Mental Wellness Companion

> *You don't have to carry this alone.*

An AI-powered mental wellness chatbot for students and anyone navigating stress, anxiety, burnout, or loneliness — available anytime, judgment-free.

---

## 🎯 The Problem

1 in 3 college students experience significant anxiety or depression. Most never get help — because therapy is expensive, waitlists are long, and at 2AM when everything feels too heavy, there's no one to call.

**MindEase fills that gap** — a compassionate AI companion that listens, supports, and guides users toward calm, available 24/7 at zero cost.

---

## ✨ Features

| Feature | Description |
|---|---|
| 💬 Just Talk | A safe, judgment-free space to share what's on your mind |
| 🧘 Calm Me Down | Guided breathing and grounding exercises for anxiety |
| 📓 Journal Prompt | Thoughtful prompts to help process emotions through writing |
| 💡 Coping Tips | Evidence-based strategies for stress, burnout, and overwhelm |
| 🌙 Sleep Help | Bedtime routines and relaxation techniques for better rest |
| 😊 Mood Check-in | Track how you're feeling throughout the day |
| 📊 Mood History | Visual tracker showing your emotional patterns |
| 🆘 Crisis Resources | Always-visible links to 988, Crisis Text Line, and global helplines |

---

## 🛠 Tech Stack

- **Frontend**: Streamlit
- **AI**: Anthropic Claude API (claude-sonnet-4-6)
- **Language**: Python 3.10+
- **Deployment**: Streamlit Cloud (free)

---

## 🚀 Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/mindease
cd mindease
pip install -r requirements.txt
export ANTHROPIC_API_KEY="your-key-here"
streamlit run app.py
```

Get your free API key at: https://console.anthropic.com

---

## ☁️ Deploy Free

1. Push to GitHub
2. Go to share.streamlit.io
3. Connect your repo
4. Add `ANTHROPIC_API_KEY` in Secrets
5. Deploy — live in 2 minutes

## ⚠️ Disclaimer

MindEase is an AI companion, not a licensed therapist or medical professional. It is not a substitute for professional mental health care. If you are in crisis, please contact:
- **988 Suicide & Crisis Lifeline** — call or text 988
- **Crisis Text Line** — text HOME to 741741
- **International resources** — findahelpline.com
